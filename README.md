Gameboxd is a mobile app that lets users track, rate, and keep a personal journal of video games they play.

The goal of the project is to build a clean, mobile-first experience focused on personal game tracking, inspired by movie journals but designed specifically for games.

---

##  Features (current)

-  Search real video games using the RAWG API
- View game details (cover, release date, rating)
- Add games to a personal journal
- Local persistence using AsyncStorage
- Built with React Native + Expo

---

## Tech Stack

- **React Native**
- **Expo**
- **JavaScript**
- **RAWG Video Games Database API**
- **AsyncStorage**
- **React Navigation**
